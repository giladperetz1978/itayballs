יוצר היילייטס כדורסל אוטומטי

פרויקט שמזהה את הכדור באוויר מתוך סרטון משחק ומרכיב אוטומטית קטעים של זריקות, כדי לייצר וידאו היילייטס קצר ונקי בעזרת YOLO ו-Streamlit.

לריפו ב-GitHub האתר החי

חשוב: דף זה הוא דף מידע. עריכת הוידאו מתבצעת באפליקציית Streamlit של הפרויקט (בהרצה מקומית או בפריסת Streamlit Cloud), ולא בתוך GitHub Pages.

איך זה עובד

המערכת עוברת על הפריימים, מזהה את מיקום הכדור, ומאתרת נקודות שיא בתנועה פרבולית כדי לסמן זריקות.

מה מקבלים

וידאו מקוצר עם רגעי הקליעות בלבד, כולל אפשרות צפייה והורדה ישירה מהאפליקציה.

סטטוס פריסה

האתר מתפרסם אוטומטית דרך GitHub Pages בכל push לענף main.
